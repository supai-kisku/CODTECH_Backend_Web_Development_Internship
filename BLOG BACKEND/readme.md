Task 1: Blog Backend System
📝 Project Overview
This project focuses on building a secure, relational backend architecture for a content management and blogging platform. The system establishes a functional API layer handling complex relational database structures to manage user identities, publish markdown/text posts, and maintain nested commenting hierarchies safely.  
⚙️ Core Architecture & Concepts

User Authentication & Session Security: Implements user enrollment, login pipelines, and route-guarding mechanisms to ensure users can only modify or delete their own posts and comments.  
Relational Database Design: Normalizes schemas using foreign-key relationships to seamlessly map user accounts to individual blog posts and associate posts with their respective comments.
CRUD Endpoint Operations: Exposes standardized RESTful interfaces for reading public content streams, creating entries, updating published materials, and deleting database records.

Technical Stack (Suggested)
Framework Engine: Django REST Framework (Python) or Express.js (Node.js)  
Database Management: PostgreSQL / MySQL
